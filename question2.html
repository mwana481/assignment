Course Name: Operating System Concepts

Course Description:

The course provides students with the basic principles of an operating system (file systems, memory management, and process management) and practical implementation of the theoretical concepts using any of the existing operating system of choice (e.g. Linux, UNIX and Windows).

Learning Outcome:
By completing the course, the student should be able to:
  • Identify Operating system structures such as components, services, system calls, system programs, and virtual machines
  • Describe different file systems features such as access methods, directory structure, protection and management.
  • Describe Memory management techniques such as Address binding, dynamic loading, dynamic linking, overlays, logical versus physical address space, paging, and segmentation.
  • Explain Process and thread management, creation, operation, scheduling; inter-process communication, and comparison between processes and threads
  • Explain Critical section problem and semaphores algorithm to solve that problem.
